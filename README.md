# Compile commands

you can try compile with this command:

```g++ main.cpp -o main -O1 -Wno-missing-braces -I raylib-5.0_win64_mingw-w64/include -L raylib-5.0_win64_mingw-w64/lib -lraylib -lopengl32 -lgdi32 -lwinmm```
